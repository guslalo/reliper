LINTERNA DE MANO STREAMLIGTH SIEGE

COD: 44943 LINTERNA DE MANO ULTRACOMPACTA Y ALCALINA

Una linterna resistente y compacta para cuando necesite una gran cantidad de luz sin requerir demasiado espacio. La Siege AA usa pilas alcalinas AA asequibles y fáciles de encontrar para ofrecer 360° de luz suave y uniforme que ilumina un área extensa.

Varios modos:

LED blanco (un LED C4®) con la tapa quitada:

Alto, para la luz más brillante: 200 lúmenes y 7 horas de funcionamiento Medio, para una luz brillante y mayor duración: 100 lúmenes, 15.5 horas de funcionamiento

Bajo, para tiempos de funcionamiento prolongados cuando se necesita menos luz: 50 lúmenes y 37 horas de funcionamiento.

LED rojo (dos LED rojos)

LED rojo, Alto (modo para visión nocturna): 0.7 lúmenes, 192 horas de funcionamiento (8 días)

Modo SOS intermitente, para señalar emergencias: 0.7 lúmenes, 288 horas de funcionamiento (12 días)
